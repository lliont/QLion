# QLion
Resources for the QLion Gold card for Sinclair QL

The QXL.WIN file from WIN_image.zip in the Win_image directory must be written to a fresly FAT32 formatted HC SD card ( 4GB to 32GB partition ).

You'll find a Users Guide in the doc directory



![QLion Gold](https://github.com/user-attachments/assets/068718e4-98b5-4f7b-a557-45dae55bf14b)


If you want to update the firmware be carefull of the cheap USB Blaster clones that work only for windows 7.
The usb blaster should be based on the FTDI's FT245+ IC to work well with the quartus drivers.
